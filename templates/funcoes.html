Funções e Procedimentos

Funções são blocos de código que executam uma tarefa específica e podem retornar um valor. Procedimentos são funções que não retornam valor, apenas executam ações. Em Python, todas as funções são definidas com a palavra-chave def.

Usar funções ajuda a organizar o código, evitar repetições e facilitar a manutenção.

Exemplo em Python - Função que retorna um valor:


def soma(a, b):
    return a + b

resultado = soma(3, 5)
print("Soma:", resultado)

Resultado esperado:

Soma: 8

Exemplo em Python - Procedimento (função sem retorno explícito):


def imprimir_mensagem():
    print("Olá, este é um procedimento.")

imprimir_mensagem()

Resultado esperado:

Olá, este é um procedimento.
