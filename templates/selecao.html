Estruturas de Seleção

Estruturas de seleção são usadas em programação para tomar decisões, executando diferentes blocos de código dependendo de condições específicas. Elas permitem que o programa escolha caminhos alternativos, tornando-o mais dinâmico e inteligente.

As estruturas de seleção mais comuns são o if, else e elif (ou else if em outras linguagens). Com elas, você pode verificar condições e decidir qual trecho do código será executado.

Exemplo em Python:


idade = 18

if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

Resultado esperado:

Você é maior de idade.
